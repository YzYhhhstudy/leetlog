# LeetLog Privacy Policy / 隐私政策

_Last updated: 2026-07-04_

## English

**TL;DR: LeetLog never sends your data anywhere except your own computer.**

- **What the extension observes** (only on `leetcode.com` / `leetcode.cn` problem pages):
  a signal that you started typing in the code editor (no keystroke content is recorded),
  the problem's public metadata (title, difficulty, tags, link), the code you submit,
  and the judge results (Accepted/Wrong Answer, runtime, memory).
- **Where it goes**: exclusively to a bridge running on **your own machine**
  (`127.0.0.1`, the LeetLog Python server or the LeetLog Bridge Obsidian plugin),
  which writes Markdown notes into your local note folder.
- **What we collect**: nothing. There is no analytics, no telemetry, no third-party server,
  no account, and no sale or sharing of data. The developers never see your data.
- **Storage**: all data lives in files on your device, under your control. Uninstalling the
  extension and plugin stops all processing; your notes remain yours.
- **Open source**: the complete source code is auditable at
  https://github.com/YzYhhhstudy/leetlog

Contact: open an issue at https://github.com/YzYhhhstudy/leetlog/issues

## 中文

**一句话：LeetLog 不会把你的任何数据发送到你自己电脑以外的地方。**

- **扩展观察的内容**（仅限 `leetcode.com` / `leetcode.cn` 题目页）：你开始在代码编辑器打字的
  信号（不记录按键内容）、题目的公开信息（标题、难度、标签、链接）、你提交的代码、
  判题结果（通过与否、运行时间、内存）。
- **数据去向**：只发送到**你自己电脑**上的桥接程序（`127.0.0.1`，LeetLog Python 服务或
  LeetLog Bridge Obsidian 插件），由它写成本地 Markdown 笔记。
- **我们收集什么**：什么都不收集。没有统计埋点、没有遥测、没有第三方服务器、不需要账号，
  更不存在出售或共享数据。开发者永远看不到你的数据。
- **存储**：所有数据都是你设备上的文件，完全由你掌控。卸载扩展和插件即停止一切处理，
  笔记仍归你所有。
- **开源**：完整源代码可在 https://github.com/YzYhhhstudy/leetlog 审计。
